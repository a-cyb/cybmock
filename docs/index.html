CYBMOCK

塞伯坦前端数据模拟服务器

快速上手 →

快速便捷

零配置、快速构建MOCK服务器，支持修改MOCK数据的热更新、支持同时启动多个MOCK服务。

简单易用

简单的规则方便配置接口数据，模拟数据支持对象、数组、函数，以及支持Express的路由规则。

高效可靠

支持使用ES Module组织和编写MOCK数据，简化复杂项目MOCK数据的开发和维护工作。

# 快速开始

# 安装
npm install -g cybmock # 或者 yarn global add cybmock

# 启动MOCK服务器
cybmock start
